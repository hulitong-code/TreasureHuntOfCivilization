# 🏛️ 文明求索 (Civilization Quest)

> 一款面向初中生的世界文明题材教育策略游戏

## 📖 项目简介

**文明求索**是一款将知识学习与策略游戏完美结合的网页游戏。玩家通过回答历史、地理、物理等学科的知识问题来扩张领土、发展文明，在游戏中自然地学习知识。

### 核心理念

- **学习即战力** — 掌握知识才能解锁能力，而非氪金
- **寓教于乐** — 策略游戏的乐趣与知识学习自然融合
- **公平竞技** — 知识解锁可能性，策略决定胜负

## ✨ 功能特性

### 已实现 (MVP)
- 🎮 文明选择系统（罗马、中华、埃及、希腊）
- 🗺️ 六边形世界地图与领土系统
- 📚 知识问答系统（历史、地理、物理、数学）
- 🏆 资源管理与回合制经营
- 💡 答题解锁领土扩张

### 规划中

- ⚔️ 文明间战斗系统
- 👥 多人在线对战
- 📖 更丰富的题库（化学、生物、经济）

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 9+ 或 pnpm

### 安装依赖

```bash
# 安装所有依赖
npm install
```

### 启动开发服务器

```bash
# 同时启动前端和后端
npm run dev

# 或分别启动
npm run dev:client  # 前端: http://localhost:3000
npm run dev:server  # 后端: http://localhost:3001
```

### 访问游戏

打开浏览器访问: **http://localhost:3000**

## 🎮 游戏玩法

### 1. 选择文明
每个文明都有独特的加成和特色单位：
- 🏛️ **罗马** — 道路建造成本-50%
- 🏯 **中华** — 科技研究速度+20%
- 🔺 **埃及** — 奇观建造速度+30%
- 🏺 **希腊** — 文化值产出+25%

### 2. 扩张领土
点击相邻的领土进行扩张，但需要先回答知识问题：
- ✅ 答对：成功占领该领土，获得资源产出
- ❌ 答错：扩张失败，回合继续

### 3. 知识挑战
随时可以进行额外的知识挑战来获得科技点和文化值：
- 📚 **历史** — 各文明的兴衰、人物、战争、发明
- 🌍 **地理** — 地形、河流、气候、资源分布
- ⚡ **物理** — 力学、能量、简单机械原理
- 🔢 **数学** — 几何、代数、数学史

### 4. 资源管理
- 🌾 **粮食** — 维持人口增长
- ⚒️ **生产力** — 建造建筑和单位
- 💰 **金币** — 购买和维护
- 🔬 **科技点** — 解锁新科技
- 🎭 **文化值** — 扩展影响力
